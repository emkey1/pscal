## 2023-10-25 - Terminal Accessory Keyboard Clicks
**Learning:** Custom keyboard accessory views (`UIInputView` used as `inputAccessoryView`) in iOS do not automatically play keyboard click sounds. Even if standard buttons are used, tapping them is silent, breaking the expected native typing experience.
**Action:** The custom `UIInputView` subclass must explicitly conform to `UIInputViewAudioFeedback` and return `true` for `enableInputClicksWhenVisible`. Furthermore, each button action needs to manually call `UIDevice.current.playInputClick()` upon `.touchDown` to simulate standard keyboard sounds.
